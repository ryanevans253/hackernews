# Hacker News Top 10

Hacker News Reader is a React application that fetches and displays top 10 stories from Hacker News.

## Installation

Install with npm

```bash
  git clone https://github.com/ryanevans253/hackernews.git
  npm install my-project
  cd my-project
```

To run:

```bash
npm run dev
```
